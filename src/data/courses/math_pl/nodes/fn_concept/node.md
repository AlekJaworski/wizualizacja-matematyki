Funkcja to jedno z najważniejszych pojęć w matematyce — opisuje zależność, w której każdemu wejściu odpowiada dokładnie jedno wyjście.<br><br>Formalnie: funkcja przypisuje każdemu $x$ dokładnie jedną wartość $y = f(x)$. Dziedzina to zbiór $x$-ów, dla których funkcja jest określona. Zbiór wartości to zbiór $y$-ów, które funkcja przyjmuje.<br><br>Wykres funkcji to zbiór punktów $(x, f(x))$ na płaszczyźnie. Każda pionowa prosta przecina wykres funkcji co najwyżej raz.

<!-- example -->
Na przykład: $f(x) = 2x + 1$. Dziedzina: wszystkie liczby rzeczywiste. $f(3) = 7$, $f(0) = 1$, $f(-1) = -1$. Wykres to prosta.<br><br>Czy $y = \pm\sqrt{x}$ to funkcja? Nie — bo np. dla $x = 4$ daje $y = 2$ lub $y = -2$ (dwie wartości, więc nie jest funkcją).

<!-- en -->
A function is one of the most important concepts in mathematics — it describes a relationship in which each input corresponds to exactly one output.<br><br>Formally: a function assigns to each $x$ exactly one value $y = f(x)$. The domain is the set of $x$-values for which the function is defined. The range is the set of $y$-values that the function takes.<br><br>The graph of a function is the set of points $(x, f(x))$ in the plane. Every vertical line intersects the graph of a function at most once.

<!-- example -->
For example: $f(x) = 2x + 1$. Domain: all real numbers. $f(3) = 7$, $f(0) = 1$, $f(-1) = -1$. The graph is a line.<br><br>Is $y = \pm\sqrt{x}$ a function? No — because e.g. for $x = 4$ it gives $y = 2$ or $y = -2$ (two values, so it is not a function).
