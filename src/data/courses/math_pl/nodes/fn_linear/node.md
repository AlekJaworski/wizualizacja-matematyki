Funkcja liniowa to najprostsza funkcja: $y$ rośnie (lub maleje) o stałą wartość przy każdym kroku $x$ (zależność wprost proporcjonalna z przesunięciem) — jej wykresem jest prosta.<br><br>Postać: $f(x) = ax + b$, gdzie $a$ to współczynnik kierunkowy (nachylenie prostej), a $b$ to wyraz wolny (punkt przecięcia z osią Y).<br><br>Gdy $a > 0$ funkcja rośnie, gdy $a < 0$ maleje, gdy $a = 0$ jest stała. Miejsce zerowe: $x = \frac{-b}{a}$.

<!-- example -->
Na przykład: $f(x) = 2x - 3$. Nachylenie $a = 2$ (rośnie — na każdy 1 w prawo idzie 2 w górę).<br><br>Przecięcie z osią Y: $b = -3$, czyli punkt $(0, -3)$. Miejsce zerowe: $2x - 3 = 0 \Rightarrow x = 1{,}5$.

<!-- en -->
A linear function is the simplest function: $y$ grows (or decreases) by a constant value at each step of $x$ (a directly proportional relation with a shift) — its graph is a line.<br><br>Form: $f(x) = ax + b$, where $a$ is the slope (the inclination of the line), and $b$ is the constant term (the point where it crosses the Y-axis).<br><br>When $a > 0$ the function is increasing, when $a < 0$ it is decreasing, when $a = 0$ it is constant. Zero (root): $x = \frac{-b}{a}$.

<!-- example -->
For example: $f(x) = 2x - 3$. Slope $a = 2$ (increasing — for every 1 to the right it goes 2 up).<br><br>Y-axis intercept: $b = -3$, i.e. the point $(0, -3)$. Zero: $2x - 3 = 0 \Rightarrow x = 1.5$.
