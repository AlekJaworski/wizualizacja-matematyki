Statystyka opisowa na poziomie rozszerzonym wprowadza miary rozrzutu — mówią, jak bardzo dane są rozproszone wokół średniej.<br><br>Kluczowe pojęcia: wariancja (średnia kwadratów odchyleń od średniej), odchylenie standardowe ($\sigma = \sqrt{\text{wariancja}}$), kwartyle (dzielą dane na cztery równe części), wykresy pudełkowe (box plot).<br><br>Im większe odchylenie standardowe, tym bardziej dane są rozproszone. Małe $\sigma$ oznacza, że dane skupiają się blisko średniej.

<!-- example -->
Na przykład: dane $\{2, 4, 4, 4, 5, 5, 7, 9\}$. Średnia $= 5$.<br><br>Odchylenia od średniej: $-3, -1, -1, -1, 0, 0, 2, 4$. Kwadraty: $9, 1, 1, 1, 0, 0, 4, 16$.<br><br>Wariancja $= \frac{32}{8} = 4$. Odchylenie standardowe $= \sqrt{4} = 2$.

<!-- en -->
Descriptive statistics at the extended level introduces measures of spread — they describe how widely the data are scattered around the mean.<br><br>Key concepts: variance (mean of squared deviations from the mean), standard deviation ($\sigma = \sqrt{\text{variance}}$), quartiles (split the data into four equal parts), box plots.<br><br>The larger the standard deviation, the more spread out the data are. A small $\sigma$ means that the data cluster near the mean.

<!-- example -->
For example: data $\{2, 4, 4, 4, 5, 5, 7, 9\}$. Mean $= 5$.<br><br>Deviations from the mean: $-3, -1, -1, -1, 0, 0, 2, 4$. Squares: $9, 1, 1, 1, 0, 0, 4, 16$.<br><br>Variance $= \frac{32}{8} = 4$. Standard deviation $= \sqrt{4} = 2$.
