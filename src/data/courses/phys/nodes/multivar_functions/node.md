A function of several variables takes more than one input and returns one (or more) output. Temperature on a map: $T(x, y)$. Height on terrain: $h(x, y)$. Spacetime metric: $g(t, x, y, z)$.<br><br>Visualizing one is harder than the 1D case — you can't draw $f(x, y)$ on a single graph the way you can $f(x)$. Workarounds: contour plots (lines of constant $f$), heatmaps, 3D surfaces. All physics worth doing is multivariable, so building intuition here is non-optional.

<!-- example -->
The temperature in a room is a function $T(x, y, z)$. At the door $T = 18°$C; at the window $T = 22°$C; near the radiator $T = 25°$C. Three numbers in, one number out — that's a scalar field. Multivariable functions describe *fields* in physics.

<!-- mistakes -->
- **"It's just like 1D, with more letters"** — the *behavior* is richer. A 1D function increases or decreases; a 2D function can have saddle points, ridges, valleys. New phenomena emerge.
- **Confusing "value of the function" with "one of the inputs"** — $T(x, y)$ has $x$ and $y$ as inputs and $T$ as output. The output is the temperature; $x$ and $y$ are coordinates.
